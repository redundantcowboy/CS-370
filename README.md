To begin this project, I was given the code for the maze itself, TreasureMaze.py, the file that saves past moves for training, GameExperience.py, and some starter code that included:
- show(qmaze)
- completion_check()
- build_model()
- play_game()
- train_step()

The main part of the code that I created myself was the qtrain() function. This function is in charge of the AI model training, which controls how the pirate explores and learns from its surroundings. More specifically, I wrote code that let the pirate try random moves or use what it learned from past moves, save those moves, and train the network.
